Reference Tabs Docker

Open reference images in docker tabs, with zooming, scrolling, mirroring, rotating, and color sampling. Supports drag and drop of multiple images.

File Menu

Open: Adds a tab and prompts to load an image.
Load: Prompts to load an image in the current tab.
Add Tab: Adds an empty tab.
Close Tabs...:
- Close Current Tab: Closes the current tab.
- Close All Tabs: Closes all open tabs.
- Close Tabs to the Left: Closes all tabs to the left of the current tab.
- Close Tabs to the Right: Closes all tabs to the right of the current tab.

Image Settings Menu

Contains per-image-tab settings.
Zoom Setting: Set the behavior of the "Fit to page" button to one of the following:
* Fit page (Default)
* Fit width
* Fit height
* Zoom 100%
Scaling Mode Setting: Whether the image should be smooth or sharp when zoomed in.
* Smooth scaling (Default)
* Sharp scaling
Change Background Color...: Set the view's background color, useful for images with transparency.

Bottom toolbar

Below the image is a toolbar. This contains:
* Zoom spinbox
* Fit to page button
* Vertical mirror button
* Horizontal mirror button
* Rotation spinbox
* Color sampler button: Click the button, then click on the image to sample while dragging. Release the mouse button to end sampling. It's not pixel-perfect.

Key+mouse shortcuts

Hold down the key combination, and click+drag the mouse.
Note: This feature is unpolished, especially Rotate mode.
* Pan mode: Space
* Zoom mode: Ctrl+Space
* Rotate mode: Shift+Space
